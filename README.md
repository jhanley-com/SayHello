# Introduction

This is a test package to learn how to publish to Packagist

## Install

Via Composer

``` bash
$ composer require Jhanley/SayHello
```

## Usage

``` php
$client = new Jhanley\SayHello();
echo $client->SayHello();
```

## Testing

``` bash
$ phpunit
```
